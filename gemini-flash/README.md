# Cat Clicker: Cosmic Evolution

Cat themed, Cookie clicker style game with visual progression and random events.

## Hook
Feed the skinny kitten treats to watch it grow into a majestic Cosmic God Cat!

## Environment
- MCP servers detected: fal-ai, chrome-devtools
- Image-gen MCP: fal-ai ✓
- Browser MCP: chrome-devtools ✓
- Pre-flight: PASSED

## Controls
- Mouse Click: Click the cat!
- Keyboard: Shortcuts for upgrades (1-4)
- Keyboard (~): Toggle FPS overlay

## Features
- **Visual Progression**: The cat grows from a skinny kitten to a majestic god cat as your score increases.
- **Treat Frenzy**: Rare golden treats fall from the sky. Click them for a 5x multiplier frenzy!
- **Auto-Defense**: Unlock the Laser Pointer to automatically repel the treat-stealing dog.
- **Multiplier Upgrades**: Buy catnip to permanently increase your score multiplier.

## How to run
1. `make dev`
2. Open `http://localhost:3214`
