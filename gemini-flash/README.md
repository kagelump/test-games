# Cat Clicker

Cat themed, Cookie clicker style game.

## Hook
Click the fat cat to earn treats, buy upgrades, and become the ultimate cat hoarder!

- MCP servers detected: fal-ai, chrome-devtools

- Mouse Click: Click the cat!
- Keyboard: Shortcuts for upgrades (1-2)
- Keyboard (~): Toggle FPS overlay

1. `make dev`
2. Open `http://localhost:3214`
